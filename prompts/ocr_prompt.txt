# OCR 提示词（占位）

> 此文件为 P0 占位，结构与字段已规划，具体提示词在 P4 阶段填充。
> 热重载：app/services/prompt_loader.py 会在 mtime 变化时自动重新读取。

## 输入
- 一份 PDF/DOCX 试卷经 OCR 后的纯文本（可能含乱码、错位、缺图）
- 图片按出现顺序以占位符 `[IMG:n]` 形式插入

## 输出（Markdown 结构）
```
# 试卷标题

## 一、选择题

### 1. 题干
[IMG:1]
A. ...
B. ...

### 2. ...

## 二、填空题
...
```

## 要求
1. 严格按试卷原貌排版，不合并、不删改、不补全
2. 数学公式保留为 LaTeX 源码（行内 $...$ / 块级 $$...$$）
3. 图片位置以 `[IMG:n]` 占位，n 与上传文件清单中的图片一一对应
4. 题目之间空一行，便于后续按题号切片
5. 答卷与试卷分页：答案放在 `# 答案` 章节，解析放在 `# 解析` 章节
